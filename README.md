# 📊 People's Priorities - AI Dashboard

## 🌍 Overview
This Streamlit dashboard helps analyze citizen feedback using AI.  
It supports multilingual translation, voice and photo input, sentiment analysis, clustering of themes, recommendations, and predictive hotspot detection.

---

## 🚀 Features
- 📝 **Sentiment Analysis** – Detects positive/negative feedback using DistilBERT.
- 🌐 **Multilingual Translation** – Translates Hindi/other languages into English via Deep Translator.
- 🔗 **Recurring Themes Analyzer** – Identifies common issues using TF‑IDF + KMeans clustering.
- 🎤 **Voice Feedback Input** – Captures spoken feedback in Hindi/English.
- 📷 **Photo Feedback Input** – Uploads images and processes them with OpenCV.
- 🤖 **AI Executive Summary** – Generates concise summaries of citizen issues.
- 📈 **MP Dashboard** – Interactive bar and pie charts of feedback data.
- ⭐ **Recommendation Engine** – Ranks projects by urgency and frequency.
- 💬 **Copilot Chatbot** – Embedding + similarity search with summarizer answers.
- 🔮 **Predictive AI Hotspot Indicator** – Detects escalation risks in complaints.
